# COMPUTER VISION

## Коммерческий опыт

- В Mamba разработал CV-модель для модерации пользовательских фотографий.
  Решение автоматизировало более 80% ручных проверок, заменило внешние модели и
  достигло качества 0.85+ F1-score.

## Проектный опыт

- В бакалаврском дипломном проекте разработал и обучил CNN-модель для
  классификации семи эмоций по изображениям лиц. Для онлайн-инференса реализовал
  FastAPI-сервис.

- В магистерском дипломном проекте разработал on-prem CV-решение для
  промышленности. Система считывает показания разных типов датчиков с помощью
  камер: цифровых, стрелочных и термометров. Решение уже применяется на
  производстве.

## Сильные стороны в CV

- Умею доводить CV-модели до production-сценариев, где важны качество,
  надежность, понятные метрики и интеграция в существующие процессы.

- Есть опыт работы с задачами классификации изображений, модерации контента и
  промышленного компьютерного зрения.

- Использую PyTorch для обучения и дообучения нейросетей, FastAPI для
  онлайн-инференса, Docker для упаковки сервисов и Prometheus/Grafana для
  мониторинга.
