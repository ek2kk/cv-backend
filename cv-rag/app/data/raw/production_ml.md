# PRODUCTION ML

## Общий подход

Разрабатываю ML- и LLM-системы полного цикла: от постановки задачи, анализа
данных и выбора модели до деплоя, мониторинга, оценки качества и оценки
бизнес-эффекта. Для меня production ML — это не только обучение модели, но и
интеграция в продукт, надежный инференс, метрики, эксплуатация и понятная
ценность для бизнеса.

## Production-задачи

- Внедрил LLM-агента для dating-copilot в Mamba, который используется в продукте
  с аудиторией 200+ тыс пользователей.

- Реализовал RAG-чат-ботов для поддержки, снизив нагрузку на саппорт в несколько
  раз и заменив внешние решения внутренними сервисами.

- Разработал CV-модель для модерации пользовательских фотографий, которая
  автоматизировала более 80% ручных проверок и достигла 0.85+ F1-score.

- Внедрил LLM-пайплайн категоризации пользовательского контента, сократив срок
  подготовки обучающих датасетов более чем в два раза.

- Реализовывал мониторинг операционных метрик и автоматические оповещения об
  аномалиях для EDNA-саппорта.

## Инструменты

- Backend и API: Python, FastAPI.

- ML/DL: PyTorch, scikit-learn, CatBoost, transformers, sentence-transformers.

- Инференс: vLLM, Triton Inference Server.

- Данные и пайплайны: SQL, ClickHouse, pandas, NumPy, Airflow, Kafka.

- Инфраструктура и эксплуатация: Docker, Docker Compose, nginx, Prometheus,
  Grafana, Alertmanager, Metabase.
