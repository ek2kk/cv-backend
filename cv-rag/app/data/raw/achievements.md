# ДОСТИЖЕНИЯ

## Ключевые коммерческие результаты

- Разработал и внедрил LLM-агента для dating-copilot в Mamba. Агент анализирует
  совместимость пользователей и помогает им в общении. Решение используется в
  продукте с аудиторией 200+ тыс пользователей.

- Реализовал RAG-чат-ботов для автоматизации работы поддержки. Решения снизили
  нагрузку на саппорт в несколько раз и позволили заменить внешние инструменты
  внутренними ML/LLM-сервисами.

- Разработал CV-модель для модерации пользовательских фотографий. Модель
  автоматизировала более 80% ручных проверок, заменила внешние модели и достигла
  качества 0.85+ F1-score.

- Внедрил LLM-пайплайн для автоматической категоризации пользовательского
  контента. Это сократило срок подготовки обучающих датасетов более чем в два
  раза.

- Строил интерактивные дашборды в Metabase для визуализации ключевых продуктовых
  и операционных метрик. Работал с SQL и большими массивами данных объемом
  10+ млн строк.

- Оптимизировал работу EDNA-саппорта: реализовал мониторинг эффективности
  сотрудников и автоматические оповещения об аномалиях в метриках.

## Production ML и инфраструктура

- Доводил ML- и LLM-решения от постановки задачи до внедрения в production:
  проектировал пайплайны, готовил данные, обучал или настраивал модели,
  разворачивал сервисы и оценивал эффект для бизнеса.

- Работал с production-стеком для ML-сервисов: Python, PyTorch, FastAPI, vLLM,
  Docker, SQL, ClickHouse, Kafka, Airflow, Grafana, Prometheus, Alertmanager,
  Metabase и nginx.

- Создавал решения, которые заменяли внешние сервисы внутренними ML-системами.
  Это повышало контроль над качеством, стоимостью, данными и интеграцией в
  продуктовые процессы.

## Проектные достижения

- В бакалаврском дипломном проекте разработал CNN-модель для распознавания семи
  эмоций по изображениям лиц и реализовал FastAPI-сервис для онлайн-инференса.

- Реализовал Telegram-бота для ежедневного мониторинга метрик онлайн-кинотеатра:
  CAC, LTV и досматриваемость. Аналитика строилась на данных из ClickHouse
  объемом около 2 млн событий.

- В магистерском дипломном проекте разработал on-prem CV-решение для
  промышленности, которое считывает показания разных типов датчиков с помощью
  камер. Решение уже применяется на производстве.
